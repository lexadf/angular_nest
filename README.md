## Getting Started

1. Clone the repository:

```bash
git clone <repository-url>
cd angular_nest
```

2. Start the development environment:

- Start the application:

```bash
docker compose up
```

- Rebuild and start:

```bash
docker compose up --build
```

- Stop the application:

```bash
docker compose down
```

## Notes

- The UI runs on port 4200
- The BFF runs on port 3000
